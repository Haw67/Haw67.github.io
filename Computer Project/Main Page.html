Welcome to NestAR, the innovation mobile application for interior design driven by augmented reality! With the help of our app, you can see and alter your room in real time, experimenting with moving, deleting, scaling, and rotating furniture. Creating your ideal space has never been simpler thanks to an intuitive interface. Use the power of augmented reality to bring your ideas to life.




No. Features Description
1 Select User allows select the models through the scroll view furniture library.
2 Place User allows place the model after select and tap on the AR Plane detected.
3 Move User allows hold the model and drag to accomplish move the model Use one finger, but it only can drag and move two second, if you want to move again need to release model then do it again the hold and drag.
4 Rotate User allows rotate the model left and right 360 degrees .But need use one finger hold the model more than 2 seconds, then swipe left or right to accomplish rotate.
5 Scale User allows to scale model use two fingers tap and hold the model do zoom in and out to accomplish scaling model.
6 Remove User allows to remove the model. When user tap the model, the top right corner of screen, will pop out a remove icon to accomplish remove the model. Once user click the remove icon will immediately remove without confirm message. But user still can select and place the model again.
7 Delete all User allows to delete all the model use a Red delete icon. When user click the red delete button, will pop out a warning message , to remind and let user confirm delete all model or cancel. But user still can select and place the model again.
8 View View toggle is use to hide and show the UI (tick mean show; untick mean hide), because there have many UI on the screen can effect the visualization. The UI hide and show include: AR Plane, AR ruler, wall color slidertoggle with color picker, scroll view furniture model library, delete icon, remove icon, and screenshot icon.
9 AR Ruler When user need to use AR ruler, they need to switch on the slidetoggle to green color, switch off color is white, and the application start this feature is off. After switch on, user can tap on the AR Plane with two point, then it will show the two red point, a yellow line and a fade in and out text write out the what is the distances in cm with 2 seconds. After two seconds the text, point and line will automatic clear (fade out).
10 Screenshot User allows to screenshot the design/ the models they interact with real-time environment. This features show then the view is untick, then the screenshot icon will show on the right middle of screen. The screenshot picture is save at the device gallery. When screenshot successful, will show a text "Screenshot Successfully!!!".
11 Light Estimation This feature is a automatic feature, it adjust the light stimulation through user device cam. When at the light places, the models bright a bit; when dark places, the models dark a bit. This feature also let the model show the soft shadow under the model.
12 Wall color paint This wall color change also can known as change the AR Plane color, because the wall is detected by AR Plane. The color is change use UI color picker. It has a slidertoggle to on off show this feature.
